One-time setup for Claude CLI
  1. Make sure claude is on your PATH.
  2. Open Terminal, cd into the Nuntius folder (where you did Load unpacked).
  3. Run this command — it wires Chrome's native-messaging host to this extension ID:
    ./native-host/install.sh …
  4. Reload Nuntius in chrome://extensions.
Ollama setup
  1. Install Ollama: ollama.com/download.
  2. Pull a model in Terminal:
    ollama pull qwen3:8b
  3. Make sure Ollama is running (the menu-bar icon on macOS).
  4. Click ↻ above to load the model list.
